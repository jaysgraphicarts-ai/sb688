## Purpose

This paperwork defines the public-safe intake for a business, artist, brand, or client entering the SB688 system.

A Business Brick describes the business operation. It is used to prepare the Memory Chip, access pass, AVA session rules, and approved Clip Bricks.

## Access model

```text
JGA holds the room.
The customer holds the access pass.
The Memory Chip links to the customer's private operation.
Clip Bricks serve the operation after verification.
```

---

## Owner approval rules

High-risk actions should be held for owner approval.

Examples:

- booking approval
- payout preparation
- contract-related action
- customer dispute response
- public campaign launch
- price changes
- sensitive memory updates

---

## Public boundary

Do not place customer secrets, passwords, private customer records, card secrets, API keys, or live Memory Chip data in this public paperwork.
