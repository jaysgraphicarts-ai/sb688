# Owner's Room Advertisement Prompt Center

Purpose: create one owner-controlled center for creating, editing, approving, publishing, and sending advertisements.

## Supported advertising work

- email advertisements
- social media posts
- social media ads
- video ad scripts
- video ad storyboards
- short-form video prompts
- image ad prompts
- product/service promotions
- campaign calendars
- captions
- hashtags
- landing page copy
- customer follow-up messages
- brand voice templates

## Owner's Room law

No advertisement publishes or sends without approval state.

AI may draft, edit, rewrite, generate concepts, compare versions, and prepare publishing packages.

AI may not publish, send, spend ad money, impersonate people, make unsupported claims, use private customer data without permission, or bypass owner review.

## Ad workflow

capture idea -> create prompt -> draft ad -> edit versions -> compliance scan -> owner approval -> schedule or publish -> log -> report -> tie off

## Email advertisement workflow

idea -> subject lines -> preview text -> body copy -> offer -> call to action -> audience list check -> unsubscribe/compliance check -> owner approval -> send queue -> delivery report

## Video ad workflow

idea -> hook -> script -> scene list -> visual prompt -> voiceover prompt -> caption prompt -> edit notes -> approval -> export package -> publish queue

## Social media workflow

idea -> platform selection -> caption -> image/video prompt -> hashtags -> CTA -> compliance scan -> approval -> schedule -> publish log

## Prompt templates

### Email ad prompt
Create a persuasive but honest email advertisement for [business] promoting [offer] to [audience]. Include subject lines, preview text, body copy, CTA, and compliance notes. Do not make unsupported claims.

### Video ad prompt
Create a short video advertisement concept for [business] promoting [offer]. Include hook, scenes, voiceover, on-screen text, visual style, CTA, and editing notes. Keep claims verifiable.

### Social post prompt
Create a social media post for [platform] promoting [offer] to [audience]. Include caption, image/video prompt, hashtags, CTA, and risk notes.

## Publishing protection

Publishing requires:
- selected business Brick
- selected platform
- final ad content
- media assets attached
- compliance scan complete
- owner approval
- schedule or send target
- ledger entry

## Truth boundary

The system helps create and organize advertising. It does not guarantee sales, platform approval, viral reach, legal compliance, tax outcomes, or ad account acceptance.

Every claim must be verifiable. Every send or publish action must be logged.
