# Work Room Brick

Purpose: one owner-controlled workspace for AI chats, files, pictures, web pages, tests, and Brick coding.

Connected AI slots:
- ChatGPT
- Grok
- Gemini
- Perplexity
- future AI adapters

Law:
- no AI touches the Spine
- all outside AI output is proposal-only until verified
- all files and pictures are stored with tenant/owner scope, hash, source, and review state
- every task enters the task queue
- every change must sync across GitHub, Docker, VS Code, and Supabase where applicable

Rooms:
- AI Desk: compare AI answers and keep source labels
- File Vault: store docs, images, exports, and attachments
- Web Lab: create and review web pages
- Test Lab: run boundary and runtime tests
- Brick Forge: code, package, clip, and review Bricks
- Owner Review: approve, reject, quarantine, or tie off changes

Records:
- work_room_items
- work_room_files
- work_room_ai_sessions
- work_room_pages
- work_room_tests
- work_room_code_bricks

Completion rule: nothing is marked complete without proof, limitation, ledger entry, and tie-off checkpoint.
