## Core law

```text
No active state becomes trusted state without verification.
```

SB688 separates operations, memory, and verification so business actions do not become trusted system state until they pass checks.

## Clean ownership model

```text
JGA holds the room.
The customer holds the access pass.
The Memory Chip links to the customer's own business operation.
The Clip Bricks they rent monthly or own outright serve that operation.
The STITCH performs the handshake.
AVA connects only after access is verified.
VERA checks important actions.
The ledger records the session.
```

Shared architecture does not mean shared private memory.
